# AI Language Card Tool
此專案是一個利用 AI 技術自動化生成語言學習資料的應用。

## 技術架構 (Technical Stack)
- **Frontend**: Streamlit
- **LLM**: Google Gemini 1.5 Flash API
- **NLP**: gTTS (Google Text-to-Speech)

## 如何啟動
1. 確保已安裝 `pip install -r requirements.txt`
2. 設置環境變數 `export GOOGLE_API_KEY="你的金鑰"`
3. 執行 `streamlit run app.py`
